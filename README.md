# Credit Risk Modeling and Stress Testing of Bank Loan Portfolios Using Machine Learning

## Overview

This project develops a machine learning-based credit risk assessment framework for World Bank-funded development projects in India. It integrates loan-level financial data with macroeconomic indicators to estimate Probability of Default (PD) and Expected Credit Loss (ECL) under baseline and stressed economic scenarios.

## Objectives

* Predict credit default risk using machine learning techniques.
* Conduct macroeconomic stress testing using GDP and CPI shocks.
* Estimate Expected Credit Loss (ECL) for loan portfolios.
* Identify sectoral vulnerabilities and risk concentrations.

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Key Findings

* Energy & Power, Urban Development & Housing, and Technology & Innovation exhibited the highest stressed default probabilities.
* Expected Credit Loss increased slightly under stressed scenarios, with Energy & Power contributing the largest portfolio risk.
* The framework demonstrates how macroeconomic stress testing can support forward-looking credit risk management and sectoral risk assessment.
